# dbms

## Build
```
cmake -S . -B build
cmake --build build
```

## Run
After you build the sources, run the following command.
```
./build/src/main
```

## Test
After you build the sources, run the following command.
```
cd build/src && ctest
```

## Development
You should install `cmake, gcc` to compile, and install the vscode extension for C++.
