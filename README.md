# dbms

## Build
```
./x build
```

## Run
```
./x run main
```

## Test
The following command runs all test.
```
./x test
```

If you want to run test in a file, you can use the following command.
```
./x test sample_lib
```

## Development
You should install `cmake, gcc` to compile, and install the vscode extension for C++.
